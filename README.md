# UnifiedMLDemo
тест




библиотеки:

pandas - библа для работы с данными

numpy - библа для мат операций

matplotlib - библа для графической интерпретации

фреймворки ml:

sklearn - классик ml

xgb(XGBoost) - библа градиентного бустинга

фреймворки dl:

pytorch - нейронные сети

Keras - высокоуровневый api для нейросетей

big data инструмены:

hadoop - хранить и обрабатывать большие данные

greenpulm - крутая субд

архитектуры нейросетей:

RNN(recurent neural network)

LSTM (Long Short-Term Memory) - улучшенная RNN с долгой памятью

GRU(Gated Recurrent Unit) - упрощённая версия LSTM

Transformers - архитектура на основе механизма внимания

Классический ML(скоринги):

скоринг - система баллов для оценки риска
